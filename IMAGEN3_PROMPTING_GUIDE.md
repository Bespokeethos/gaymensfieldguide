## CORE AESTHETIC: NANO BANANA

**Keywords to ALWAYS include:**
- `industrial paper texture`
- `dark background`
- `neon yellow / banana accent`
- `monospace terminal font`
- `tactical, survival, authentic`
- `technical overlay text`
- `circuit board traces`
- `glitch effects`

**Keywords to AVOID:**
- `clean`, `minimal`, `modern`
- `corporate`, `professional`
- `stock photo`, `realistic`
- `3D render` (unless specified)

---

## PROMPT STRUCTURE

```
[SUBJECT]: [Description of main element]
[STYLE]: Nano Banana aesthetic - dark industrial paper texture, neon yellow accents
[ELEMENTS]: Technical annotations, monospace font, circuit traces
[MOOD]: Tactical, Survival, Authentic, Industrial
[FORMAT]: PNG, optimized, dark mode friendly
```

## IMAGEN 3 SPECIFIC TIPS

1. **Be Explicit**: Imagen 3 responds well to detailed descriptions
2. **Layer Instructions**: Subject → Style → Elements → Mood
3. **Negative Prompts**: Use "avoid" or "no" for unwanted elements
4. **Aspect Ratio**: Specify (e.g., "16:9", "square", "portrait")
5. **Quality**: Add "high detail", "sharp", "optimized PNG"
